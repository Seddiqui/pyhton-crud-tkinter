# pyhton-crud-tkinter
A simple Python CRUD app Using Tkinter UI Library and SQLite Database.


# How to run
run the command. 
`
git clone https://github.com/shtayeb/pyhton-crud-tkinter.git
`

go to project directory
`cd python-crud-tkinter
`

run the command
`python ms.py`
or
`py ms.py`
